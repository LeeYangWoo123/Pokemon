# 🎮 Pokémon GO Type Effectiveness Calculator

Flet을 이용해 만든 포켓몬 GO 전용 타입 상성 계산기입니다. 복합 타입의 약점과 내성을 실시간으로 계산해줍니다.

## ✨ 주요 기능
- **복합 타입 계산:** 두 가지 타입을 선택하면 배율을 곱산하여 최종 상성을 도출합니다.
- **포켓몬 GO 기준 적용:** 원작과 다른 포켓몬 GO만의 배율(0.39배 등)을 완벽히 적용했습니다.
- **반응형 UI:** 깔끔한 다크 모드 인터페이스와 애니메이션 아이콘을 제공합니다.

## 🛠 기술 스택
- **Python** (Core Logic)
- **Flet** (Cross-platform UI)
